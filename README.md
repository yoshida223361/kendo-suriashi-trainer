# 剣道すり足ランダム指示

iPhoneのSafariで使う前提の、前後左右ランダム指示Webアプリです。

## 使い方

1. `index.html` をブラウザで開く。
2. 最短秒数・最長秒数・練習時間・合図方式などを設定する。
3. 「開始」を押す。
4. 表示された方向にすり足する。

## iPhoneでアプリ風に使う

GitHub Pages / Netlify / Vercel などにアップロードしてURLで開き、Safariの共有メニューから「ホーム画面に追加」してください。

## 主な機能

- ランダム秒数ごとの前後左右指示
- 色と巨大表示による方向指示
- 体育用ホイッスル風の指示音
- 音あり / 音なしの切り替え
- 前後多め、同方向連続なし
- 開始カウントダウン：なし / 3秒 / 5秒 / 10秒
- 開始・終了は長いホイッスル、方向指示は短いホイッスル
- 練習時間設定
- PWA用 manifest / service worker 付き

## 注意

iPhoneでは、音はユーザーが「開始」や「音テスト」を押した後に鳴ります。練習中に画面が消える場合は、自動ロック設定を長めにしてください。
